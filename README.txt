GitHub Viewer to aplikacja internetowa, która umożliwia wyszukiwanie użytkowników GitHuba oraz przeglądanie ich repozytoriów. Wyświetla również informacje na o liczbie commitów, gwiazdek oraz komentarzy.

Jak uruchomić?
1. Pobierz pliki źródłowe(app.js, styles.css, index.html).
2. Otwórz plik index.html w dowolnej przeglądarce.
3. W polu tekstowym wpisz nazwę użytkownika GitHuba, a następnie kliknij "Search"
4. Po kliknięciu przycisku "Search", aplikacja wyświetli informacje o użytkowniku oraz liste repozytorów. Po naciśnięciu w repozytorium zostaniemy do niego przeniesieni na zewnętrzną stronę. 




GitHub Viewer is a web application that allows you to search for GitHub users and view their repositories. It also displays information on the number of commits, stars and comments.

How to get started.
1. Download the source files(app.js, styles.css, index.html).
2. Open the index.html file in any browser.
3. Type your GitHub username in the text box, then click "Search".
4. After clicking "Search", the application will display user information and a list of repositories. After tapping on the repository, you will be taken to it on an external page.
